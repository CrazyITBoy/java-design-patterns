## Intent
实现了非类型语言的灵活性保住了语言的安全 

![alt text](./etc/abstract-document.png "Abstract Document Traits and Domain")


## 适应的情况
什么时候去使用抽象文档设计模式？

* 需要动态的添加新的属性
* 你需要一种灵活的反思
* 让系统的耦合性更低
